Resumen de Etiquetas HTML

Curso: ISW-512 Diseño de Aplicaciones Web
Carrera: Ingeniería del Software – ISW
Autor ; Juan carlos Ulate Cardenal

A. Etiquetas de estructura básica

  • <!DOCTYPE>: Define el tipo de documento (HTML5).
  • <html>: Raíz del documento HTML.
  • <head>: Encabezado con metadatos.
  • <title>: Título que aparece en la pestaña del navegador.
  • <base>: URL base para enlaces relativos.
  • <link>: Vínculo a recursos externos (CSS, íconos).
  • <meta>: Metadatos (autor, descripción, charset, etc).
  • <body>: Contenido visible de la página.

B. Etiquetas de estructura de sección

  • <h1> a <h6>: Encabezados jerárquicos.
  • <hgroup>: Agrupa encabezados.
  • <address>: Información de contacto.
  • <div>: División genérica para estructurar contenido.
  • <iframe>: Muestra otra página dentro de la página.

C. Etiquetas de estructura de grupo

  • <ul>, <ol>, <li>: Listas desordenadas/ordenadas.
  • <dl>, <dt>, <dd>: Lista de definiciones.
  • <figure> y <figcaption>: Figura ilustrativa con leyenda.
  • <details> y <summary>: Información colapsable.
  • <dialog>: Diálogo emergente.

D. Etiquetas de texto

  • <p>, <br>, <pre>: Párrafos, saltos de línea y texto preformateado.
  • <blockquote>, <q>, <cite>: Citas.
  • <abbr>, <dfn>: Abreviaturas y definiciones.
  • <code>, <kbd>: Código fuente y entrada de teclado.
  • <b>, <strong>, <i>, <em>, <mark>: Formato de texto.
  • <small>, <del>, <ins>, <sub>, <sup>: Texto especial.
  • <time>, <data>, <span>: Datos temporales o personalizados.

E. Etiquetas de enlace

  • <a>: Enlace a otra página o sección.
  • <area>: Área clicable en un mapa de imagen.
  • <map>: Define un mapa de imagen.

F. Etiquetas de imagen

  • <img>: Inserta una imagen.
  • <picture>, <source>: Imagen adaptable según pantalla.
  • <canvas>: Área de dibujo para gráficos dinámicos.
  • <svg>: Gráficos vectoriales escalables.

G. Etiquetas multimedia

  • <audio>, <video>: Reproducción de archivos multimedia.
  • <source>: Fuente alternativa para audio o video.
  • <track>: Subtítulos y descripciones.
  • <canvas> y <svg>: También se usan para animaciones o juegos.
  • <iframe>: Para incrustar videos (por ejemplo, de YouTube).
